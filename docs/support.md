## What To Include In A Support Request

To help diagnose issues quickly, include:

- Slack workspace name
- Slack workspace/team ID (if available)
- Command used (for example `/leaderboard flickle`)
- Approximate time of issue (UTC preferred)
- Screenshot or copied error message

## Common Setup Steps

1. Install the app to your Slack workspace.
2. Run `/settings` as a workspace admin/owner.
3. Select the public channels the bot should monitor.
4. Post supported game result messages in those channels.
5. Use `/leaderboard` and `/leaderboard-monthly` to view results.

## Supported Commands

- `/settings`
- `/leaderboard framed`
- `/leaderboard framed-one`
- `/leaderboard flickle`
- `/leaderboard guess-the-game`
- `/leaderboard-monthly`
- `/leaderboard-monthly framed`
- `/leaderboard-monthly framed-one`
- `/leaderboard-monthly flickle`
- `/leaderboard-monthly guess-the-game`

## Data Deletion Requests

Workspace admins may request data deletion. Include workspace/team details and admin confirmation in your request.
